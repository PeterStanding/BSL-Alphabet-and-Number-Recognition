# BSL Sign Detecion
A tool that can be used to detect BSL Sign Language Numbers and Alphabet signs.
I have learnt a great deal from creating this, such as:
- Image Recognitiion
- Pickle and training models
- Hand Landmarks

Fundamentally it works, and does recognise the different BSL Alphabet characters, just need to improve the accuracy of the project. As sometimes it misreads the sign

All useful hyperlinks for the project can be found within the 'HyperLink.txt' file in the directory. The only thing I have not included is the dataset, but that can be found in the link below 👍

Once the dataset has been downloaded, it needs to be saved in the same folder as 'main.py' otherwise the model links on lines 10 and 11 will need to be altered

Dataset can be found here 👇

https://www.kaggle.com/datasets/erentatepe/bsl-numbers-and-alphabet-hand-position-for-mediapipe?resource=download
